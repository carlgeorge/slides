Packages

A compressed archive that contains files and metadata.

The files are usually for a specific application or library.

Typical files include compiled binaries, scripts, and documentation.

Installing a package places those files in the appropriate locations in the filesystem.

Package Formats

Metadata

A package's metadata describes several properties.

  • name
  • description
  • version
  • release
  • relationships to other packages

Relationships

  • requires
  • conflicts
  • provides
  • obsoletes

Package Mangement

Package managers are software tools that allow you to install and upgrade packages from remote sources.

Of course, they also allow you to remove packages.

Because they parse and interpret package metadata, they allow you to perform package transactions in a consistent, reliable manner.

Package Managers

  • yum
  • apt
  • pacman
  • zypper

Repositories

A remotely accessible collection of packages.

They also contains summary metadata about the packages in the repository.

A repository can be first party (from the distribution vendor) or third party.
